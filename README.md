# rock-paper-scissors-game
Developed an interactive Rock Paper Scissors game using HTML, CSS, and JavaScript. The application features real-time user interaction, play against the computer and randomized computer moves, and dynamic score tracking, and reset the game anytime with a dedicated reset button.Implemented core game logic with efficient condition handling and DOM manipulation to update results instantly, with a clean and responsive user interface.

## Live Demo
https://nazuk-kabra1111.github.io/rock-paper-scissors-game/

## Features

*  Interactive gameplay(click-based controls)
*  Randomized computer moves
*  Real-time score tracking(User vs Computer)
*  Dynamic result display(Win/Lose/Tie)
*  Clean and responsive UI using Flexbox

## Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & Layout (Flexbox)
* **JavaScript** – Game Logic & DOM Manipulation

## How It Works
* User selects **Rock, Paper, or Scissors**
* Computer randomly generates a choice (using inbuit library- MATH functions)
* Game logic compares both choices:

  * Rock beats Scissors
  * Paper beats Rock
  * Scissors beats Paper
* Score updates instantly based on the result
